# First-party skills in aios-kit

Only skills whose source of truth is `aios-kit` itself should live here.

Currently, LLL is intentionally not vendored here because it is an independent first-party project: `LinLin00000000/lins-living-loop`.

Hermes-local or `.agents`-local skills can be listed in `skillpack.yaml` as first-party local paths until they are extracted into their own repos or into this directory.
